# Quiz App

A simple and interactive quiz application built with Flutter.

## Features
- Interactive quiz session with multiple choice questions.
- Score calculation at the end of the quiz.
- Fully responsive across Web, Desktop, and Mobile.

## Setup
Run the app using the standard Flutter toolchain:
```bash
flutter run
```
